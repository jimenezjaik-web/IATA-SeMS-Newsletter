# IATA SeMS Newsletter

This repository contains a responsive HTML email newsletter created as a portfolio sample focused on promoting Safety Management System (SeMS) content.

The project follows email-safe HTML and CSS practices, ensuring compatibility across major email clients while maintaining a clear and professional content structure.

## Live Preview
👉 https://jimenezjaik-web.github.io/IATA-SeMS-Newsletter/

## Key Features
- Table-based layout for maximum email-client compatibility
- Responsive structure optimized for desktop and mobile viewing
- Clear and structured content hierarchy
- Externally hosted images for reliable email delivery
- Readable typography and well-organized sections

## Technical Notes
- Built specifically for HTML email (not standard web layouts)
- Inline and email-safe CSS practices applied
- Designed to handle limited CSS support across email clients
- Graceful degradation when images are disabled

## Use Case
This project represents a corporate or informational newsletter used to communicate SeMS-related updates, insights, or announcements. It demonstrates practical HTML email development skills for real-world communication scenarios.

📌 **Note:** Rendering may vary across email clients due to differences in CSS support and client-specific behavior.
